[`../concepts/`](../concepts/) and [`../survey/`](../survey/).

   — a design-philosophy blueprint: the AutoResearch paradigm, the layered

2. [`2026-06-deep-research-2-recommendation.md`](./2026-06-deep-research-2-recommendation.md)
